#Get Started
###Prerequisites
1. Git
2. Ruby
  - [Windows](http://rubyinstaller.org/)
  - [Everyone else](https://www.ruby-lang.org/en/)

###Run
1. gem install sass
2. gem install scss_lint
3. npm install -g gulp
3. npm install
4. gulp
