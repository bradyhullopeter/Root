#Get Started
###Prerequisites
1. Git
2. Ruby
  - [Windows](http://rubyinstaller.org/)
  - [Everyone else](https://www.ruby-lang.org/en/)

###Run
1. gem install sass scss_lint
2. npm install -g gulp karma karma-cli protractor
3. npm install
4. gulp
